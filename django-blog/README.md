# 🚀 Aplicação de Blog

Este projeto faz parte da minha trilha de estudos no repositório **learning-path**. O objetivo aqui foi aprender a criar uma aplicação web, em Django, totalmente funcional: modelos, templates, visualizações e URLs. Entender como o Django funciona e como os diferentes componentes do framework interagem

## 📋 Descrição do Projeto
Uma breve explicação do que o projeto faz. 
> "Este é uma aplicação de blog simples que permite aos usuários navegar por todas as postagens publicadas e ler postagens individuais. Também vou criar uma interface de administração simples para gerenciar e publicar postagens."

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.13
* **Bibliotecas:** `django`, `os`
* **Ferramentas:** VS Code

## 🧠 O que eu aprendi (O que estou aprendendo)
* Como projetar os modelos de dados;
* Como aplicar migrações ao banco de dados;
* Como implementar paginação de objetos;
* Como construir views baseadas em classes;
* Como criar formulários para permitir que seus usuários recomendam postagens por e-mail e comentem em postagens.
